# Kanban Board (React + Zustand)

Project: simple board of tasks with Dnd

## Технологии

- React + TypeScript
- Zustand
- React DnD
- Feature-Sliced Design

## Функциональность

- jeck
- Просмотр задач
- Перетаскивание задач между колонками
- Добавление и удаление задач

## Структура

Проект построен на FSD (features/entities/widgets)

## Запуск

```bash
npm install
npm run api
npm run dev
```
